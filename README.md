# Portfolio Website

A modern, responsive personal portfolio website showcasing projects, skills, and experience.

## Features

- **Responsive Design** - Adapts seamlessly to mobile, tablet, and desktop
- **3D Background** - Interactive particle constellation effect
- **Smooth Animations** - Scroll-triggered reveals and smooth transitions
- **Project Showcases** - Detailed modals with image carousels
- **2-Color Theme** - Clean pitch black and cyan aesthetic

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, Animations)
- JavaScript (Three.js for 3D effects)
- Responsive Grid Layouts
