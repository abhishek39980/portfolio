Self-Balancing Robot

Technologies: C++, Arduino Uno, PID Control, Sensor Fusion

This project is a sophisticated autonomous system designed to maintain vertical stability while navigating its environment. By combining real-time sensor fusion with wireless connectivity, it demonstrates complex control theory and hardware integration.


Core Functionality

  • Vertical Stability: Uses a PID control loop with data from an MPU-9250 IMU to counteract gravity by adjusting motor speed.
  • Obstacle Avoidance: Features an HC-SR04 Ultrasonic sensor to detect objects and trigger avoidance maneuvers.
  • Connectivity: Integrated Wi-Fi + Bluetooth module allows for real-time data telemetry.

Hardware Architecture

  • Top Tier: Ultrasonic sensor for clear sight lines and communication modules.
  • Middle Tier: Arduino Uno, MPU-9250 IMU, and power distribution.
  • Base Tier: L298N Motor Driver and high-torque BO motors on rubber wheels.
